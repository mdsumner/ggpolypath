ggpolypath
----------

A `ggplot2` geom for polygons with holes, called 'geom\_polypath'.

Install
-------

Install from CRAN.

``` r
install.packages("ggpolypath")
```

Install the development version.

``` r
## install from Github
devtools::install_github("mdsumner/ggpolypath")
```

-------

``` r
vignette("ggpolypath")
```
