ggpolypath

A ggplot2 geom for polygons with holes, called 'geom_polypath'.

Install

Install from CRAN.

install.packages("ggpolypath")

Install the development version.

## install from Github
devtools::install_github("mdsumner/ggpolypath")

Example

See the vignette for examples.

vignette("ggpolypath")

Gory details
